# Telegram Bot с Веб-Интерфейсом

Проект включает в себя Telegram бота и веб-интерфейс для управления.

## Быстрый старт

1. Клонировать репозиторий:
\`\`\`bash
git clone https://github.com/Lerwix/tg_bot.git
cd tg_bot
\`\`\`

2. Установить зависимости:
\`\`\`bash
pip install -r requirements.txt
\`\`\`

3. Настроить конфигурацию:
\`\`\`bash
cp config.example.py config.py
# Отредактируйте config.py, добавив ваш Telegram токен
\`\`\`

4. Запустить бота:
\`\`\`bash
python main.py
\`\`\`

5. Запустить веб-сервер:
\`\`\`bash
python server.py
\`\`\`

## Веб-интерфейс

Веб-интерфейс доступен по адресу: https://lerwix.github.io/tg_bot

### Локальный доступ:
- Откройте браузер
- Перейдите по адресу: http://localhost:5000
